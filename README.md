# 火星第一眼：1976 維京一號的拍照任務｜數位任務頁

國小高年級彈性學習課程（科技教育／資訊教育議題）之課堂互動工具，共 5 節課。
以 1976 年維京一號（Viking 1）拍下人類第一張火星地表照片的史實為情境，
帶學生走過「記錄 → 壓縮 → 編碼 → 傳送 → 還原」的完整旅程。

## 使用方式

啟用 GitHub Pages 後，開啟網站首頁即會進入「任務入口」，選擇各節任務即可使用；
所有頁面皆為單一 HTML 檔、免安裝、離線下載亦可使用（瀏覽器直接開啟）。
